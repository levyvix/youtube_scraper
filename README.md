# YouTube Scraper


Esse script foi desenvolvido para obter as informações dos vídeos do Lead e dos prospects.

Informações como:
- Canal
- Título do vídeo
- Data de publicação
- Número de visualizações
- Número de likes
- Número de comentários

Esse script obtem 2 tipos de informações dos canais:
- Videos Recentes
- Vídeos Em Alta
